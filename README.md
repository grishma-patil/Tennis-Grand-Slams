This project provides an analytical journey through the history of Men's Singles Grand Slam tournaments from 1968 to 2023. The analysis uncovers fascinating trends, player dominance, and rivalries in the tennis world, with a special focus on the legendary "Big 3": Roger Federer, Rafael Nadal, and Novak Djokovic.<br>
<br>
Using advanced data visualization and analysis techniques, this project captures the milestones, achievements, and rivalries that have shaped tennis history.<br>
<br>

Global Insights: Visualized Grand Slam-winning countries using interactive maps.<br>

Rivalries: Network diagrams and treemaps reveal intense rivalries among top players like Nadal vs. Djokovic and Federer vs. Djokovic.<br>
Timeline of Titles: Depicted the gripping race for Grand Slam titles among the Big 3.<br>
